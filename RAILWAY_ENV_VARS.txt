IMPORTANT NOTES:
- PORT uses Railway's special variable ${{RAILWAY_PUBLIC_PORT}}
- CLIENT_URL will be updated later with your Vercel frontend URL
- Make sure to set root directory to "server" during deployment
- The build command should be: npm install
- The start command should be: npm run production
